還差一步:填入 Google Maps API key

  1. 打開 src/config.js
  2. 把 GOOGLE_MAPS_API_KEY = "___" 換成你的 API key
  3. 存檔後頁面會自動重新載入

申請方式見 README「取得 Google Maps API key」。
